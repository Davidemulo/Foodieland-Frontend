# Foodieland UI

A modern food recipe website built with HTML and CSS.
This project replicates a clean UI design with multiple pages including homepage, blog, and contact.

---

## Features

* Responsive homepage layout
* Blog/article page
* Contact form UI
* Reusable components (cards, sections)
* Clean and modern design

---

## Project Structure

```
foodieland/
│── index.html
│── blog.html
│── contact.html
│── styles.css
│── assets/
│── README.md
```

---

## Tech Stack

* HTML5
* CSS3

---

## Getting Started

1. Clone the repo:

```
git clone https://github.com/Davidemulo/Foodieland-Frontend.git
```

2. Open the project:

```
cd foodieland
```

3. Run in browser:

* Open `index.html`
